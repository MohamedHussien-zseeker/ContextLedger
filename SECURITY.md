# Security Policy

## Local-first design

ContextLedger runs entirely on your machine. No data leaves your system unless you explicitly configure external services (e.g., Qdrant for semantic search).

## Reporting vulnerabilities

If you discover a security issue, please use [GitHub Security Advisories](https://github.com/MohamedHussien-zseeker/ContextLedger/security/advisories/new) to report it privately.

For non-sensitive bugs, you can also open a [GitHub issue](https://github.com/MohamedHussien-zseeker/ContextLedger/issues).

## Scope

This security policy applies to the ContextLedger Python package and CLI. It does not cover third-party services you connect to (Qdrant, custom providers, etc.).
